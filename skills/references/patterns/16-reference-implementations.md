# 多语言参考实现

每种架构在各语言社区的优秀参考实现。

## Layered Architecture

| Language | Repository | Stack |
|----------|-----------|-------|
| Java | Spring Boot + JPA standard layered | Spring Boot / JPA |
| Go | Standard Go project layout | net/http / GORM |
| Python | Django / FastAPI layered | Django / FastAPI |
